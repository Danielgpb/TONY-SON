# Core 30 SEO Implementation Plan — Tony&Son

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Align the Tony&Son Astro website with the Core 30 GBP-SEO system — updating all title tags, H1s, meta descriptions, content, schema, internal links, and FAQ sections across homepage + 2 category pages + 13 service pages.

**Architecture:** The site uses Astro 6 with centralized service data in `src/data/services.ts` that drives dynamic routes via `src/pages/fr/services/[slug].astro`. Category pages (`coiffure/index.astro`, `barbier/index.astro`) filter from this data. All changes target the French (`/fr/`) version only. The key insight: updating `services.ts` automatically updates all 13 service pages via the template.

**Tech Stack:** Astro 6, TypeScript, HTML, CSS, JSON-LD Schema
